All algorithms to live by

2019-06-14 012:30:45

Design and algorithms are the backbone to the efficiency and coherency of a program. we've learned that algorithms come in various fashions. We have 5 different types of algorithms: linear, logarithmic, quadratic, constant, and polylogarithmic. Each have different facets and speed efficiencies. The most basic understanding is that in bigger applications you want a program that uses either logarithmic or linear or constant algorithms due to the speed at which they are run. Linear moves upward in processing speed as input increases, logarithmic starts slow but as input increase so does the processing speed, and constant keep the same processing speed with whatever amount of input. Quadratic is the slowest due to the number of iterations needed to be done to complete one task, therefore as input increases the processing speed exponentially increases.
